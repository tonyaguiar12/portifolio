BlinkFund — Web3 Crowdfunding Platform on Solana

Next.js 16 • Supabase • Prisma • Upstash Redis • Tailwind CSS
Next.js 16 Supabase Prisma Upstash Redis Tailwind
O que eu construí

Criei a BlinkFund — uma plataforma de crowdfunding Web3 pensada para doações em SOL, integrada com o ecossistema Solana. Arquitectei o backend e o frontend com foco em fiabilidade e experiência do utilizador, usando Next.js no frontend e Supabase + Prisma no backend, com cache em Upstash Redis.

  • Projetei o fluxo de tokenomics e o processo de doação em SOL.
  • Implementei autenticação passwordless (SIWS) e suporte PWA para acesso móvel.
  • Integrei notificações e serviços externos para faturação e seguimento de doações.

O site está deployado em blinkfund.fun/pt. Ver site

Seven Faith — App Mobile de Fé e Bem-Estar

React Native • Expo • LLaMA/Groq
React Native Expo LLaMA / Groq i18n
O que eu fiz

Desenvolvi a app do zero com React Native e Expo, focando-me na experiência do utilizador e na retenção através de gamificação. Construi o fluxo de criação de hábitos, o sistema de pontos e feedback háptico para reforçar ações positivas.

  • Implementei a lógica de gamificação e o sistema de recompensas.
  • Integrei um motor de IA (LLaMA/Groq) para gerar sugestões personalizadas e quizzes adaptativos.
  • Adicionei internacionalização (PT, EN, ES, FR) e persistência local com AsyncStorage.

ShopVerde — Backend para E-commerce Sustentável

Laravel • PHP • MySQL • JWT/Sanctum
Laravel PHP MySQL JWT / Sanctum
O que implementei

Implementei o backend do ShopVerde, uma API RESTful preparada para cargas reais de um e-commerce. Trabalhei na modelação da base de dados, endpoints seguros para gestão de produtos, encomendas e utilizadores, e integrei gateways de pagamento.

  • Autenticação e autorização com JWT / Laravel Sanctum.
  • Integração com PayPal e Stripe e envio de notificações por e-mail.
  • Notificações em tempo real por WebSockets para estados de encomenda.
