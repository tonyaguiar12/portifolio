BlinkFund — Web3 Crowdfunding Platform on Solana

Next.js 16 • Supabase • Prisma • Upstash Redis • Tailwind CSS
Next.js 16 Supabase Prisma Upstash Redis Tailwind
Resumo

Plataforma fullstack de crowdfunding Web3 que permite arrecadar doações em SOL diretamente pelo feed do X/Twitter via Solana Blinks. Arquitetura com Next.js 16, Prisma e Supabase (Postgres + Realtime), cache em Upstash Redis e UI em Tailwind CSS com Framer Motion.

  • Sistema de tokenomics e fluxo de doações em SOL.
  • Autenticação passwordless (SIWS) e PWA.
  • Integração com serviços externos para faturação e notificações.

Seven Faith — App Mobile de Fé e Bem-Estar

React Native • Expo • LLaMA/Groq
React Native Expo LLaMA / Groq i18n
Resumo

App mobile com sistema de gamificação e feedback háptico para criar e reforçar hábitos diários. Integração de IA generativa (Groq / LLaMA 3.3) como conselheiro emocional, histórico de conversas e quizzes adaptativos.

  • Gamificação e feedback háptico.
  • Internacionalização (PT, EN, ES, FR).
  • Leitor bíblico otimizado e persistência local com AsyncStorage.

ShopVerde — Backend para E-commerce Sustentável

Laravel • PHP • MySQL • JWT/Sanctum
Laravel PHP MySQL JWT / Sanctum
Resumo

Desenvolvi uma API RESTful para e-commerce com autenticação segura, gestão de produtos, categorias, utilizadores e encomendas. Integrações com gateways de pagamento (PayPal, Stripe), serviços de logística, e notificações (e-mail e WebSockets) para atualizações em tempo real.

  • Modelação da base de dados e aplicação de padrões (MVC, Repositories).
  • Segurança e escalabilidade pensadas desde a arquitetura.
