Sobre Mim

Sempre gostei de perceber como as coisas funcionam por trás do ecrã e, acima de tudo, de resolver os problemas que aparecem no caminho. Sou um programador web prático e autónomo. No meu dia a dia, sinto-me confortável a construir interfaces rápidas com React e Next.js, a montar backends com Laravel ou a criar soluções em WordPress. Não me prendo a uma única ferramenta: se o projeto precisar de uma app, pego no React Native; se for preciso automatizar uma tarefa chata, escrevo um script em Python. O meu foco é escrever código limpo, entregar projetos que realmente funcionem e garantir que quem usa do outro lado tem uma boa experiência.
